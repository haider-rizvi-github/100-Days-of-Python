# Day 3 — Control Flow & Logical Operators

This folder contains beginner Python exercises focused on `if` statements, logical operators, and simple user interaction.

## Files

- `Odd_or_Even.py`
  - Prompts the user for a whole number.
  - Uses the modulo operator to determine whether the number is even or odd.

- `Teasure_Island_game.py`
  - A text-based adventure game.
  - The player makes choices by entering letters to navigate paths and try to find the treasure.
  - Demonstrates nested `if`/`else` logic.

- `Ticketing_system.py`
  - A rollercoaster ticket price calculator.
  - Uses age and height checks to determine ride eligibility and ticket cost.
  - Includes an optional photograph add-on.

## How to run

Open a terminal in this folder and run the desired script with Python 3:

```bash
python3 Odd_or_Even.py
python3 Teasure_Island_game.py
python3 Ticketing_system.py
```

## Notes

- Make sure to enter valid numeric input when prompted.
- The adventure game is case-insensitive for the expected choices.
- This set of programs is designed to reinforce control flow fundamentals in Python.
