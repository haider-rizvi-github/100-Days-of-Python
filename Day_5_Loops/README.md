# Day 5 - Loops

This folder contains Python exercises focused on using loops. Each script demonstrates a different looping concept or use case.

## Files

- `FizzBuzz.py`
  - Implements the classic FizzBuzz challenge.
  - Prints numbers from 1 to 100.
  - Prints `Fizz` for multiples of 3, `Buzz` for multiples of 5, and `FizzBuzz` for multiples of both.

- `Highest_score.py`
  - Demonstrates how to find the maximum value in a list.
  - First uses the built-in `max()` function.
  - Then repeats the same task using a `for` loop.

- `Loops.py`
  - Shows examples of both `for` loops and `while` loops.
  - Iterates through a list of fruits.
  - Prints numbers using `range()`.
  - Calculates totals from a list and sums numbers from 1 to 100.

- `Password_generator.py`
  - Generates a password using letters, numbers, and symbols.
  - Prompts the user for password length and counts of numbers/symbols.
  - Builds the password using random selection.

## How to run

Open a terminal in this folder and run a script with Python. For example:

```bash
python3 FizzBuzz.py
```

For `Password_generator.py`, run the script and follow the prompts.

```bash
python3 Password_generator.py
```

- These examples are part of the 100 Days of Python learning series.
- They are designed to practice loops, list operations, and basic input/output in Python.
