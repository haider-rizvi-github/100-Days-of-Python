# 100 Days of Python - Day 1 Beginner

This repository contains the first day of the "100 Days of Python" learning challenge. The focus is on Python basics: printing output, variables, and user input.

## Files

- `task.py`
  - Prints "Hello world!" to the console.
  - Demonstrates a simple script and how to run it from the shell.

- `band_name_generator.py`
  - User-interactive program.
  - Prompts for your name and your pet's name.
  - Combines input values into a suggested band name.

## What I learned

- How to write and run a Python script.
- Python syntax basics, including comments and statements.
- Using `print()` for output.
- Using `input()` to capture user input.
- String concatenation (`"Your band name could be: " + Name + " " + Pet_Name`).
- Difference between running Python from shell and Python REPL.

## Run locally

Open the terminal in this directory and run:

```bash
python3 task.py
python3 band_name_generator.py
```

## Notes

- On the Python REPL (`>>>`) you cannot run shell commands like `/usr/bin/python3 "script.py"`; do that in shell prompt.
- These were your first steps in building confidence with Python.
